# **Admin Access Instructions**

1. **Create an account on the website normally.**  
   Use the standard registration form to create a user profile.

2. **Log into MongoDB Atlas and open the `users` collection.**  
   Locate the document associated with the account you just created.

3. **Update your role from `USER` to `ADMIN`.**  
   - Find the `role` field  
   - Change the value from `"USER"` to `"ADMIN"`  
   - Click **Update** to save the change

4. **Log into the Admin Panel.**  
   Once your role is updated, you will automatically gain access to the admin dashboard.

5. **Use the Admin Panel to manage site content.**  
   As an admin, you can:
   - Create new recipes  
   - Add new ingredients  
   - Add or edit resource links  
   - Review and approve submitted community recipes  

6. **Approve submitted recipes carefully.**  
   When you approve a submitted recipe:
   - It is immediately added to the live database  
   - It becomes visible to all users  
   - No further review step exists  
   
   Make sure the recipe is complete, accurate, and appropriate before approving it.
